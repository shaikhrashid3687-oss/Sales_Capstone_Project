# Sales Analytics Capstone Project 
## Objective Analyze sales and profit data to identify trends, top products, and business insights. 
## Tools Used - Excel - Python (Pandas, Matplotlib) - SQL - Power BI 
## Project Structure - Data Cleaning - Exploratory Data Analysis - SQL Insights - Power BI Dashboard 
## Key Insights - Sales increased yearly but high discounts reduced profit - Technology category generated 
   highest profit margin - West and South regions performed best 
## Outcome Delivered a complete analytics solution with actionable business recommendations.
## Common Interview Questions (Sample Answers)

I analyzed the Superstore dataset to identify sales and profit patterns using Excel, SQL, Python, and Power BI. I cleaned the data, explored trends, wrote SQL queries for key metrics, and built an interactive Power BI dashboard to present insights.

I focused on Total Sales, Total Profit, Profit Margin, Sales by Region, and Product Category because these KPIs help identify revenue drivers and areas for improvement.

Because incorrect or inconsistent data can lead to misleading insights. Cleaning ensures accuracy and reliability.
